What is this?

This program uses a simple genetic algorithm to evolve random two-wheeled shapes into cars over generations. Based on BoxCar2D, but written from scratch using the same physics engine (box2d).
seedrandom.js by David Bau. (thanks!)

Controls

Save Population Saves the current population to your browser.
Load Population Restores a previously saved population.
Hide Cars Toggles car rendering — simulation runs faster without drawing.
New Population Keeps the generated track and respawns all cars from scratch.
Create New World From Seed The same seed always generates the same track — agree on a seed with friends and compete! :)
Mutation Rate The chance that each gene of each organism will change to a random value when a new generation is created.
Mutation Size The range by which a gene can mutate. Smaller values mean the gene will stay closer to its original value.
Elite Clones Number of the best cars that will be copied unchanged into the next generation.
▶ Best Result Stops the current simulation and replays the best car. Click again to resume.

Graph

🔴 Red Best score in each generation
🟢 Green Average of top 10 cars in each generation

Genome

The genome consists of:

  • Wheel count (1 gene, range 1–8)
  • Chassis shape (8 genes, 1 per vertex)
  • Wheel size (8 genes, 1 per wheel)
  • Wheel position (8 genes, 1 per wheel)
  • Wheel density (8 genes, 1 per wheel) — darker = denser
  • Chassis density (1 gene) — darker = denser

Notes

The simulation is not fully deterministic, so your best car may perform slightly differently on a replay. Terrain gets harder with distance.
If something looks wrong — refresh the page.
